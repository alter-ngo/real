# Instructions to run the backend

_NB: you will need python 3 & pip installed_

- go into `/backend` folder

- activate the environment `source venv/bin/activate`

- install all packages required `pip install -r requirements.txt` (if needed)

- run `python manage.py runserver` to run the backend server

- if migrations are needed run `python manage.py makemigrations` and `python manage.py migrate`

- go to `http://localhost:8000/admin` to see admin page

\*If you need help with setting up a python environment use this resource:

##### https://sourabhbajaj.com/mac-setup/Python/virtualenv.html
